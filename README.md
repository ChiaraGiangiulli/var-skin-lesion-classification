# VAR - Skin Lesion Classification

Project for the **Artificial Vision and Recognition (VAR)** course, focused on the problem of skin lesion classification using Computer Vision techniques.

## 📌 Project Objective
The main focus of this work is to develop and compare two different classification methodologies:

1. **Classical Approach:** extraction of hand-crafted visual features (related to color, shape, and texture) followed by the training of a traditional classifier (e.g., SVM, Random Forest, or KNN).
2. **Deep Learning Approach:** development and training of a Convolutional Neural Network (CNN) to delegate automatic feature extraction and end-to-end classification to the model.

## 📊 Dataset
The project utilizes the **HAM10000** dataset, a widely recognized medical archive in the dermatological field.
- **Size:** ~10,000 dermatoscopic images.
- **Task:** Multiclass classification across **7 different categories** of skin lesions.
- **Source:** [Kaggle - Skin Lesion Segmentation and Classification](https://www.kaggle.com/datasets/surajghuwalewala/ham1000-segmentation-and-classification)
